# Fashion Brand Email

This template was created for a clothing brand's email campaign using email code best practices.

## Highlights
• mso | (IE) used for:
    background image
    + 2 columns
    + 3 columns.

• Columns stack depending on device width. Mobile friendly.

• Table based coding (one div for the box-shadow).
